# Prompt Builder & Reference

A single-page web tool designed to streamline the creation of structured AI prompts using established frameworks and advanced prompting techniques.

## Features

### 1. Live Prompt Framework Builder
Assemble complex prompts in real-time using industry-standard frameworks. As you fill in the fields, the prompt is dynamically constructed in a live preview window. Blank fields are silently omitted to maintain prompt cleanliness.

**Included Frameworks:**
* **CO-STAR:** Context, Objective, Style, Tone, Audience, Response.
* **AUTOMAT:** Act as, User, Targeted action, Output, Mode, Atypical, Topic.
* **CRAFT:** Context, Role, Action, Format, Tone.
* **RACE:** Role, Action, Context, Expectation.
* **RTF:** Role, Task, Format.
* **ROSES:** Role, Objective, Scenario, Expected Solution, Steps.
* **CARE:** Context, Action, Result, Example.
* **CRISPE:** Capacity/Role, Insight, Statement, Personality, Experiment.
* **APE:** Action, Purpose, Expectation.

### 2. Technique Reference Library
A searchable, filtered repository of advanced prompting techniques to enhance AI interactions.
* **Search & Filter:** Quickly find techniques by name or difficulty level (Beginner, Intermediate, Advanced).
* **Structural Guides:** Covers everything from basic structure and delimiters to complex patterns like Chain of Thought (CoT), ReAct, and Tree of Thoughts (ToT).
* **One-Click Copy:** Each technique includes an example that can be instantly copied to your clipboard.

### 3. Modern, Responsive Interface
* **Dark Mode Aesthetic:** Built with a professional, low-strain interface using `DM Mono` and `DM Serif Display` typography.
* **Mobile Optimized:** Features a specialized bottom navigation bar for seamless use on mobile devices.
* **Zero Dependencies:** A lightweight, single-file HTML/CSS/JS solution that requires no installation or external libraries.

## Usage

1.  **Select a Framework:** Choose a framework based on your specific task (e.g., CO-STAR for reports, RTF for quick code snippets).
2.  **Input Details:** Fill out the relevant fields in the builder.
3.  **Refine:** Use the Techniques panel to find specific syntax (like delimiters or structured output requirements) to add to your prompt.
4.  **Copy & Execute:** Use the "Copy" button to grab your assembled prompt and paste it into your AI of choice.
